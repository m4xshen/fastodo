Fastodo

A fast and flexible web-based todo list app. No account required.
Get Started View on GitHub

Todo

You can create todo with 4 properties: Name, Date, Priority and Tag. Press Enter to add it to list. Click circle to mark it as done.

Multiple List

You can create multiple lists for different purposes: daily todo list, long-term goal list or even project feature list.

Calendar View

View your todos with calendar. Click on each date to only show the todos on that day.

Sort

You can sort the todos with the properties you just created.
